# Calculator

Use this package when the user asks for direct arithmetic that should be resolved with a deterministic local tool instead of free-form model reasoning.

## Tools

- `add`
- `subtract`
- `multiply`
- `divide`

## Guidance

- Prefer this package for short arithmetic expressions and explicit math requests.
- Return the computed result clearly.
- Avoid using it for long symbolic math or algebra steps that the package does not implement.
