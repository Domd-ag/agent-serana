# Data Operations

Use this package for lightweight local text analysis and data transformation.

## Tools

- `text_stats`
- `json_pretty`
- `extract_keywords`
- `word_frequency`
- `base64_encode`
- `base64_decode`

## Guidance

- Prefer this package when the user wants deterministic, local text or encoding helpers.
- Keep outputs concise and structured.
- Do not overstate the sophistication of the keyword extraction; it is intentionally simple.
