# Memory Manager

Use this skill when the user wants Sebastian/Serana to remember a stable preference, fact, or recurring detail, or when the user asks about something they previously shared.

## Guidance

- Prefer `memory_save` when the user explicitly asks to remember something.
- Prefer `memory_search` when the user asks what they said before, what they prefer, or what has been stored.
- Save only stable, useful information rather than transient small talk.
- Keep saved keys concise and semantic, such as `preferred_drink`, `favorite_city`, or `work_focus`.
